# Sales Prediction Using Python

## Overview

This project predicts product sales based on advertising budgets spent on TV, Radio, and Newspaper advertisements.

## Dataset Features

- TV Advertising Budget
- Radio Advertising Budget
- Newspaper Advertising Budget
- Sales (Target Variable)

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Machine Learning Algorithm

- Linear Regression

## Steps Performed

1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Selection
5. Model Training
6. Prediction
7. Evaluation

## Evaluation Metrics

- MAE
- MSE
- RMSE
- R² Score

## Results

Achieved approximately 88%–92% prediction accuracy using Linear Regression.
